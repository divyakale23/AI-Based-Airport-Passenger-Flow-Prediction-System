# AI-Based Airport Passenger Flow Prediction System

## Project Overview

This project forecasts passenger demand at San Francisco International Airport (SFO) using historical passenger traffic data and SARIMA time-series forecasting.

## Objectives

- Analyze historical passenger traffic
- Forecast future passenger demand
- Support airport resource planning
- Improve operational efficiency

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- SARIMA
- Streamlit

## Dashboard Features

- Future Passenger Forecast Table
- Passenger Demand Trend Graph
- Operational Insights
- Resource Planning Support

## Dataset

Source:
DataSF Air Traffic Passenger Statistics

## Author

This project was developed as part of my MBA specialization in Artificial Intelligence & Machine Learning (AI & ML) and demonstrates the practical application of data analytics, time-series forecasting, and dashboard development in the aviation sector.

The project uses historical passenger traffic data from San Francisco International Airport (SFO) to forecast future passenger demand using the SARIMA forecasting model and presents the results through an interactive Streamlit dashboard. It highlights the use of data-driven decision-making for airport resource planning and operational efficiency.

Through this project, I gained hands-on experience in:

Data Collection and Cleaning
Exploratory Data Analysis (EDA)
Time-Series Forecasting using SARIMA
Data Visualization
Dashboard Development with Streamlit
Airport Operations and Resource Planning Analytics
